# TypeScript Foundations

A collection of TypeScript examples covering core concepts used in modern web development and Playwright automation.

## Topics Covered

- Variables (`var`, `let`, `const`)
- Primitive Types (`string`, `number`, `boolean`, `null`, `undefined`)
- Arrays
- Tuples
- Type Inference
- Union Types
- Type Aliases
- Classes
- Interfaces
- Optional Properties
- Readonly Properties
- Interface Extension
- Functions
- Arrow Functions
- Generics
- Intersection Types
- `any`
- `unknown`
- `never`
